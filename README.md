## 🚀 Key Highlights

- **🧠 Production AI Showcase**: Featuring Agentic AI workflows like **AI Hospital Appointment System**, **Stubble Burning Agent**, and **AI Recruitment Automation**.
- **🌐 3D Interactive Visuals**: Custom Three.js interactive profile sphere with real-time tracking.
- **⚡ Performance First**: Optimized for speed with component lazy loading and Vite.
- **🛠️ AI Automation Specialist**: Proficient in Python, FastAPI, LangChain, Groq API, Gemini API, and building n8n workflows.

## 📦 Featured Projects

### 1. AI Hospital Appointment System
Automated scheduling system leveraging agentic AI workflows to optimize booking, reduce no-shows, and streamline patient intake.
- **Tech**: Python, FastAPI, Agentic AI

### 2. Stubble Burning Agent
AI-driven agricultural monitoring and intervention system designed to predict and mitigate stubble burning events through satellite data analysis.
- **Tech**: Python, Machine Learning, Agentic AI

### 3. AI Recruitment Automation
Fully automated recruitment pipeline with multi-criteria evaluation using Gemini LLM and real-time applicant tracking via n8n.
- **Tech**: Python, n8n, Gemini API, Gmail API, Google Sheets

### 4. Local RAG PDF Assistant
Context-aware Q&A on PDF documents with <100ms response latency using Groq's ultra-fast LLM inference and semantic search.
- **Tech**: Python, LangChain, Groq API, FastAPI, FAISS

### 5. Agentic Feedback System
Autonomous feedback analysis leveraging Gemini LLM for sentiment analysis and automated workflow execution using n8n.
- **Tech**: Python, n8n, Gemini API, Google Sheets

### 6. Chat Groq Agent + AI Email
Multi-step agent workflows achieving <100ms response time for autonomous decision-making and generating professional AI email responses.
- **Tech**: Python, Groq API, n8n, Gmail API

## ⚙️ Installation & Development

1. **Clone & Install**
   ```bash
   git clone https://github.com/Rajputritik9695/portfolio-.git
   cd portfolio-
   npm install
   ```

2. **Run Development**
   ```bash
   npm run dev
   ```

3. **Build & Optimize**
   ```bash
   npm run build
   ```
